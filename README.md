Forked Xenia-Dashboard for this. (Fork of: https://github.com/ALHROOBIX/Xenia-Dashboard)

YouTube quick guide to installation: https://www.youtube.com/watch?v=-uMGqclqpbM

This app is compatible with the UIX Emulinator for Cool People BYOND Helper app to make the shortcut files this project uses automatically. Go here for that helper app: https://www.byond.com/games/Mistamage123/UIXEmulatorinatorforCoolpeople

Process to use the BYOND helper app is as follows 1.) Download BYOND and make an account (although an account is not necessary) 2.) Go to the URL provided above and click Download 3.) BYOND pager should open. Install the app and click 'Host' 4.) Dream Daemon (dreamdaemon.exe) should open. Set security to trusted or else you'll get spammed with dialog prompts, then click GO. 5.) The world will be hosted on a closed port. Click world-->join or the yellow button to launch the app 6.) In the app's toolbar, select 'Xenia-Dashboard Settings', set your xbox 360 games directory, and make sure USE is checked off 7.) Set the app settings to your retro profile needs. 8.) Click Generate Games.Ini when you're ready to commit 9.) Open the Xenia-Dashboard.exe and set your Games folder to the same folder you specified as your xbox 360 games directory 10.) Your games are there!

NOTE: The helper app is optional. You can add entries manually by going to the xbox 360 games folder and placing shortcuts XDB-your-game-here.lnk as a base. Xenia-dashboard will successfully scan them either way.

NOTE 2: Adding Xenia paths to the app (should be) optional.

NOTE 3: If you need a video guide on using the app and the helper app, watch this YouTube video: https://www.youtube.com/watch?v=-uMGqclqpbM
